## 🖥️ Projeto

Esse é um projeto Web responsivo de uma cápsula do tempo para exibir memórias em linha do temp

## 🚀 Tecnologia

Esse projeto foi desenvolvido durante a NLW da Rocketseat com as seguintes tecnologias

- HTML
- CSS
- GIT e Github
